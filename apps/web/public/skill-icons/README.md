# Skill icon asset library

- `*.webp`: 256 × 256 runtime assets for the web app.
- `source/*.png`: high-resolution generated source assets.
- `manifest.json`: stable IDs, Chinese display names, categories, and public URLs.
- `contact-sheet.jpg`: visual index of all 30 icons.

All icons were generated individually with the built-in image generation tool. The shared art direction is a polished fantasy RPG UI badge: one centered readable symbol, a deep navy rounded-square background, a subtle beveled gold rim, painterly 2.5D lighting, no text, and no watermark.
