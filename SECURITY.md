# Security Policy

## Supported version

Agent City is currently pre-1.0. Security fixes are applied to the latest code
on the `main` branch.

## Report a vulnerability

Please use GitHub's private vulnerability reporting for this repository. Do not
open a public issue containing exploit details, personal data, access tokens,
API keys, OAuth credentials, or private file paths.

Include:

- the affected component and version or commit;
- reproduction steps;
- expected and observed impact;
- any suggested mitigation;
- whether the issue is already being exploited.

Allow maintainers time to investigate and release a fix before public
disclosure.

## Exposed credentials

If a real credential is committed or shared, revoke or rotate it immediately.
Removing it from the latest commit is not sufficient because it may remain in
history, caches, logs, or forks.
